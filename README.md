# simulator_sirkuit_digital
Game simulasi untuk tingkat SD (fase C)

Nama Aplikasi: Simulator Sirkuit Digital (Petualangan Arus Elektron)

Target Audiens: Siswa Sekolah Dasar (SD) Kelas 4–6 (Rentang Usia 9–12 Tahun). 

Fokus Pembelajaran: Siswa dapat menyimpulkan secara mandiri efek dari variasi rangkaian (Seri vs Paralel), jumlah baterai, hambatan lampu (Watt) terhadap intensitas kecerahan visual atau potensi malafungsi sirkuit (overload). 

Filosofi STEM: Mengusung model Sandbox Environment. Anak-anak diberikan kebebasan penuh memanipulasi variabel listrik secara ekstrem untuk memicu hubungan sebab-akibat (lampu meredup, menyala normal, atau meledak/putus).

Tujuan:
Mengeliminasi miskonsepsi umum anak SD bahwa "menambah baterai selalu membuat lampu lebih terang".
Menyajikan lingkungan Sandbox yang bermakna, dengan memanfaatkan teknologi untuk pendidikan.
